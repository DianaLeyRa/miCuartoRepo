# miCuartoRepo
Mi cuarto repositorio. Aprendiendo a clonar repositorios


## Como crear un repositorio remoto

1 Crear un repo en GitHub

2 Colocar nombre del repo con formato marc dawn

3 escribir una pequeña descripción del repo

4 Elegir si va a ser privado o publico
 
5 Elegir si se creara el archivo Readme

6 Crear repositorio
